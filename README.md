# esp32_touch_button
9 Touch Button ESP32.

- Button contact pad: 10 x 10 mm.
- u[0]-u[8] - Status Button: -3 No Press, 3 - Press.
- Press User Button > 3s - Reboot Chip (Watchdog Reset).
